# 🚀 DSA Visualiser

An interactive web application that helps visualize popular Data Structures and Algorithms through animations. Built using HTML, CSS, and JavaScript to make learning algorithms intuitive and engaging.

## ✨ Features

### 🔄 Sorting Visualiser
- Bubble Sort
- Selection Sort
- Insertion Sort
- Merge Sort
- Quick Sort

**Features**
- Adjustable array size
- Speed control
- Pause / Resume
- Reset
- Step-by-step animations
- Live algorithm information
- Time complexity display

---

### 🔍 Searching Visualiser
- Linear Search
- Binary Search

**Features**
- Generate random arrays
- Automatic sorting for Binary Search
- Adjustable speed
- Target value input
- Low / Mid / High pointers
- Color-coded visualization
- Pause / Resume
- Reset
- Live statistics
  - Comparisons
  - Current Index
  - Execution Time
  - Search Status
